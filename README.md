# MultiPlug.Ext.Brainboxes
Brainboxes IO Extension for the [MultiPlug](https://www.multiplug.app/) .Net Edge Computing Platform.

Use this Extension to connect to Brainboxes Ltd digital Input and Output hardware - For the full product range view at [brainboxes.com](https://www.brainboxes.com/products/remote-io).
